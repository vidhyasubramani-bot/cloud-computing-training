# Kubernetes Learning

## 📌 Overview
This repository contains my Kubernetes learning journey, including core concepts, hands-on practice, YAML manifests, and deployment examples. The goal of this repository is to understand how Kubernetes manages, scales, and orchestrates containerized applications.

- Kubernetes Architecture
- Kubernetes Cluster
- Control Plane
- Worker Node (Data Plane)

- etcd
- Scheduler
- Controller Manager
- Cloud Controller Manager
- Kubelet
- Kube Proxy
- Pods
- ReplicaSets
- Deployments
- Services
- YAML Files
- Minikube
- kubectl Commands
